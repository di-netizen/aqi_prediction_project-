# 🌫️ Air Quality Index (AQI) Prediction using Machine Learning

This project focuses on predicting the **Air Quality Index (AQI)** based on various environmental features such as PM2.5, PM10, NO₂, CO, O₃, SO₂, Temperature, and Humidity using machine learning models. The goal is to demonstrate a complete data science pipeline from data generation to deployment-ready prediction.

- PM2.5
- PM10
- NO₂
- SO₂
- CO
- O₃
- Temperature
- Humidity
